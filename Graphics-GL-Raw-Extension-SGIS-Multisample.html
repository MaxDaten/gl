glSampleMaskSGIS :: MonadIO m => GLclampf -> GLboolean -> m () Source

Usage: glSampleMaskSGIS value invert

The parameter value is a ClampedFloat32.

The parameter invert is a Boolean, one of: GL_FALSE, GL_TRUE.

This command is an alias for Graphics.GL.Raw.Extension.EXT.Multisample.glSampleMaskEXT'.
