glGetPixelTexGenParameterfvSGIS :: MonadIO m => GLenum -> Ptr GLfloat -> m () Source

Usage: glGetPixelTexGenParameterfvSGIS pname params

The parameter pname is a PixelTexGenParameterNameSGIS, one of: GL_PIXEL_FRAGMENT_ALPHA_SOURCE_SGIS, GL_PIXEL_FRAGMENT_RGB_SOURCE_SGIS.

The parameter params is a CheckedFloat32.

The length of params should be COMPSIZE(pname).
