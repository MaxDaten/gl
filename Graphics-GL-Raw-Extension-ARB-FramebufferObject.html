glBlitFramebuffer :: MonadIO m => GLint -> GLint -> GLint -> GLint -> GLint -> GLint -> GLint -> GLint -> GLbitfield -> GLenum -> m () Source

Usage: glBlitFramebuffer srcX0 srcY0 srcX1 srcY1 dstX0 dstY0 dstX1 dstY1 mask filter

The parameter mask is a ClearBufferMask, one of: GL_ACCUM_BUFFER_BIT, GL_COLOR_BUFFER_BIT, Graphics.GL.Raw.Extension.NV.CoverageSample.GL_COVERAGE_BUFFER_BIT_NV', GL_DEPTH_BUFFER_BIT, GL_STENCIL_BUFFER_BIT.
